 # Promptopia #
Promptopia is a prompts sharing web app where you can share your prompts.
## Features
* User Authentication
* Google Accounts Integration
* Database Connectivity
* Customized Search
* CRUD operation on Prompts

## Technologies Used
* Next.JS
* React
* MongoDB
* NextAuth
* Tailwind CSS
